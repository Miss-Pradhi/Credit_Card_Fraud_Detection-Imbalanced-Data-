Credit Card Fraud Detection-->
<br>
This project detects fraud in credit card transactions of Imbalanced Data using machine learning.
It includes a trained model and a Streamlit app to make predictions.

Files-->
* `Train_Model_Code.py` – trains the ML model
* `Streamlit_App.py` – Streamlit app for prediction
* `credit_fraud_data.csv` – dataset
* `requirements.txt` – needed libraries


How to Run-->
1. Install packages:
pip install -r requirements.txt

2. Run the app:
streamlit run Streamlit_App.py


Purpose-->
To identify fraudulent transactions in highly imbalanced data.
